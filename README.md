## **⚙️ How to run locally**

1. Clone this repository:  
   git clone [https://github.com/bakadja/neue-fische-ki-dev-challenges.git](https://github.com/bakadja/neue-fische-ki-dev-challenges.git)

2. Navigate to the challenge of your choice:  
   cd challenge-01-whos-online

3. Run the JavaScript file using Node.js:  
   node main.js
